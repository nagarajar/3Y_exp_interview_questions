Git Setup
----------
git config --global user.email "nagaraja.r@cesneosilica.com"
git config --global user.name "nagarajar"

To check user details
$ git config --list

Create new branch from main/master branch: When you want to start a new feature, you create a new branch off main using 
$ git branch new_branch
Once created you can then use 
$ git checkout new_branch 
to switch to that branch

If your Local branch and remote branch is the same name then you can just do it:
$  git push origin branchName

When your local and remote branch name is different then you can just do it:
$ git push origin localBranchName:remoteBranchName


Git pull from particular branch
$  git pull origin branchName
ex:
$ git pull origin master

Update remote in local as per remote
$ git branch -a
$ git remote prune origin
